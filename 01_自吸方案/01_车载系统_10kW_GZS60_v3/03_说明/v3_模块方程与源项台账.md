## 2. 主链路模块

### EnvironmentFreshAir

作用：

- 提供环境空气组成、温度、湿度和压力边界。
- 按电流和目标氧计量比给出新鲜空气流量。

源项：

```text
O2 / N2 / H2O_v 来自环境空气
H2O_l = 0
```

### EGRMixer

作用：

- 混合新鲜空气与上一拍 EGR 回流节点。

关系：

```text
m_i,out = m_i,fresh + m_i,egr
T_out, p_out = 质量加权平均
```

### Compressor

作用：

- 施加简化升压和升温边界。

关系：

```text
p_out = p_in + dp_compressor
T_out = T_in + dT_compressor
m_i,out = m_i,in
```

### Intercooler

作用：

- 施加目标出口温度和固定压降。
- 对超饱和水蒸气做冷凝裁剪。

关系：

```text
T_out = T_target
p_out = p_in - dp_intercooler
```

### HumidifierDryWetLumped

作用：

- 用上一拍湿侧入口节点向干侧传水和传热。
- 输出干侧出口节点和湿侧出口节点。

关键关系：

```text
NTUgain = hum_NTU_ref * (m_ref / m_dry)^hum_flow_exp
epsilon = 1 - exp(-NTUgain)
dry_need = max(m_vapor_sat_dry_out - m_vapor_dry_in, 0)
wet_available = max(m_vapor_wet_in, 0)
m_transfer = epsilon * min(dry_need, wet_available)
dp_dry = hum_dry_dp_ref * (m_dry / m_ref)^hum_dp_exp
dp_wet = hum_wet_dp_ref * (m_wet / m_ref)^hum_dp_exp
```

传热采用集总有效度口径：

```text
epsilon_H = 1 - exp(-UA / C_min)
dT_dry = epsilon_H * (T_wet_in - T_dry_in) * C_min / (m_dry * cp_gas)
T_dry_out = T_dry_in + dT_dry
T_wet_out = T_wet_in - dT_dry
```

当前实现补充：

- 传水源项已从纯膜阻力通量改为低维 epsilon-NTU 水容量模型；
- `hum_NTU_ref` 和 `hum_flow_exp` 控制等效传水能力随流量变化；
- `hum_mem_D_eff_m2_s` 仍保留在参数表中，但当前传水主源项不再用 `delta/Deff` 直接计算通量；
- `hum_UA_W_K` 用于干湿侧集总传热，压降仍由干湿侧参考压降和流量指数给出；
- 该模块是系统级集总换湿/换热模型，不解析膜内分布、壁面热容和沿程状态；当前用于趋势、边界和规格点一致性分析；
- 台架堆入口湿度只作系统对比，不作为车载加湿器出口硬拟合目标。

### PEMFCStackCore

作用：

- 保存阴极 O2/N2/H2O、阳极 H2/H2O 和电堆温度状态。
- 计算反应源项、膜水迁移、电压、热源和出口流量。

当前状态向量：

```text
[m_O2_ca
 m_N2_ca
 m_H2O_v_ca
 m_H2_an
 m_H2O_v_an
 T_stack]
```

液水处理边界：

- 阴极和阳极库存状态只保存水蒸气，不保存独立液水库存；
- 过饱和水蒸气通过饱和裁剪转为液化诊断量；
- 当前等效为瞬时排出或未显式跟踪的液水去向，不用于排水量、液水滞留或两相压降定量结论。

Faraday 源项：

```text
m_O2React = I*N/(4F)*M_O2
m_H2React = I*N/(2F)*M_H2
mWater    = I*N/(2F)*M_H2O
```

压力库存：

```text
pCa = pO2 + pN2 + pH2O_v
pAn = pH2 + pH2O_v
```

出口流量：

```text
mCaOutRaw = K_ca_out * max(pCaPre - pcBack, 0) / limiter
mAnOutRaw = K_an_out * max(pAnPre - panBack, 0) / limiter
```

二周目压力修正：

```text
caOut(6) = pcBack
anOut(6) = panBack
```

解释：

- `pCa_kPa / pAn_kPa` 只作为堆内库存压力诊断进入 summary。
- 下游模块读取的出口节点压力是背压边界，不是堆内库存压力。
- 阴/阳极流道体积固定为几何量级，不再作为压力拟合旋钮。

热侧阶段 A 当前补充口径：

```text
Q_gen  = I * N * (Vtn - V)
Q_cool = h_cool_eff(flow_coolant) * (T_stack - T_cool_in)
Q_amb  = h_amb * (T_stack - T_amb)
Q_gas  = m_gas_out * cp_gas * max(T_stack - T_ca_in, 0)
```

其中：

- `flow_coolant = coolant_flow_L_min`
- `h_cool_eff` 由 `thermal_stageA_cooling_flow_curve.csv` 插值给出
- `T_cool_in` 当前取台架 `coolant_inlet_temp_C`
- `C_stack_J_K` 在阶段 A 冻结
- `hum_heat_eff` 在阶段 A 冻结

当前不引入完整冷却液动态回路、泵控制、阀控制或散热器子模块。

### EGRValve

作用：

- 按 `egr_fraction_cmd` 将湿侧出口流量分成 EGR 回流和尾排预出口。

关系：

```text
m_egr = r * m_wetOut
m_vent_pre = (1-r) * m_wetOut
```

### BackPressureValveWithOutletManifold

作用：

- 将尾排端压力对齐环境压力。
- 输出背压阀压降诊断。

关系：

```text
dp_bp = max(ventPre(6) - p_amb, 0)
vent(6) = p_amb
```
