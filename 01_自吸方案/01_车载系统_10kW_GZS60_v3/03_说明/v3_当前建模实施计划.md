# v3 按《系统及其控制》重构计划

## Summary
- 已确认当前 `v3` 的主模块重构方向是：把旧的固定混合、固定背压、固定分流链路，替换成以尾气压力链为核心的低维系统模型。
- 本轮已实施的关键变更是：
- `EnvironmentFreshAir`：按“压缩机入口需求 - 实际 EGR 回流量”补入新鲜空气。
- `EGRMixer`：改为真实质量混合，不再按固定总流量和固定比例分配。
- `PEMFCStack`：电压模型已切换到新书籍 `E - V_act - V_ohm - V_conc` 结构，阴极出口流量由 `p_tailgas + dp_hum_wet + dp_separator` 约束。
- `HumidifierWet`、`Separator`：显式输出压降诊断，参与尾气压力闭合。
- `TailGasManifold`、`EGRValve`、`BackPressureValve`、`EGRReturnPipe`：替代旧 `BackPressure` / `EGRSplitter` 占位逻辑，建立低维压差驱动分流链。

## Implementation Changes
- 当前主拓扑定为：
- `EnvironmentFreshAir -> EGRMixer -> Compressor -> Intercooler -> HumidifierDry -> PEMFCStack -> HumidifierWet -> Separator -> TailGasManifold -> {EGRValve -> EGRReturnPipe -> EGRMixer, BackPressureValve -> ambient}`
- 模块间主接口仍保持 7 维阴极节点：
- `m_O2_kg_s`
- `m_N2_kg_s`
- `m_H2O_v_kg_s`
- `m_H2O_l_kg_s`
- `T_C`
- `p_kPa`
- `liquid_present`
- 尾气闭环通过两个离散反馈实现：
- `EGR_return_delay`：打断回流到混合器的代数环。
- `Stack_downstream_delay`：打断“电堆出口流量 <- 尾气压力链 <- 电堆出口流量”的代数环。
- `summary_vector` 字段表已收口到验证入口的局部字段清单。
- 当前不再保留独立 summary helper 脚本，避免脚本目录膨胀。
- `systemSummary` 当前新增核心字段已定为：
- `m_fresh_in_kg_s`
- `m_egr_return_kg_s`
- `m_vent_out_kg_s`
- `p_tailgas_kPa`
- `p_stack_downstream_kPa`
- `dp_hum_wet_kPa`
- `dp_separator_kPa`
- `dp_egr_valve_kPa`
- `dp_bp_valve_kPa`

## Calibration And Validation
- 旧无 EGR 13 点极化拟合参数和检查结果已删除。
- 当前电压模型只保留新书公式结构和未重新标定的默认系数，不能声明极化曲线已完成拟合。
- 后续需要按新书 `E/V_act/V_ohm/V_conc` 结构重新建立无 EGR 电压标定流程。
- `run_vehicle_10kw_gzs60_v3_validation.m` 当前已验证：
- 新鲜空气流量会随实际回流增加而下降。
- `xO2_ca_in`、`pO2_ca_in`、`pO2_ca` 在 EGR 工况下下降。
- 阀压降符号正确，且 `EGR` 回流前无液水。
- 物种残差和能量残差满足当前数值容差。
- 下一步仍需补强：
- 新书电压公式的无 EGR 极化重新标定。
- 13 点无 EGR 阴极压降/尾气链路定量标定 helper。
- 将电压标定与压力链误差分开记录，避免再次混用旧拟合结论。
