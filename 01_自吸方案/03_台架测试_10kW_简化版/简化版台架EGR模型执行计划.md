# 简化版台架EGR模型执行计划

## 任务边界

本任务在 `01_自吸方案\03_台架测试_10kW_简化版` 内建立新的简化版台架测试结构模型。该工程包与 `02_台架测试_10kW` 的完整台架模型、寿命模型、条件扫描脚本隔离，不沿用旧的研究脚本和验证图片。

## 建模原则

1. 以 Simulink 模型为主，脚本仅作为初始化、数据整理、参数拟合和 MATLAB 图窗展示的辅助。
2. 新建干净的简化版模型，复用已有完整电堆核心，不简化电堆内部阴阳极流道、电压、热、膜水、液态水、相变、歧管压力-流量关系和源项表达式。
3. 简化对象仅限台架/BOP外围：忽略空压机和中冷器物理模型；水分离器只去除液态水；阀门、三通只承担流量分配/经验映射功能。
4. 第一版按稳态工况点模型处理，不建立动态台架控制器。
5. 不生成 PNG 等图片文件；仿真结果展示使用 MATLAB 绘图窗口。

## 数据口径

1. 无EGR基础极化曲线来自 `10kw电堆台架稳态测试数据整理.xlsx`，并加入补充点 `20A / 0.05 A/cm2 / EGR率=0 / 单片电压=0.818V`。
2. EGR数据来自 `10kw短堆阴极尾气循环测试_0.1Acm2加20A补充_整理版_修正.xlsx`。
3. 无EGR工况读取阴极入口压力、阴极出口压力、阴极出口温度的实测值；阴极出口背压不再用 `入口压力 - 5 kPa` 的经验假设。
4. 循环比主口径优先级：
   - `回流参数_校准EGR率`
   - `回流参数_校准回流 / 回流参数_入堆流量流量计(SLPM)`
   - `(回流参数_入堆流量流量计(SLPM) - 台架供给_给气流量(SLPM)) / 回流参数_入堆流量流量计(SLPM)`
5. 所有流量读入后统一换算为模型内部单位，避免体积流量、质量流量、摩尔流量混用。
6. 标定阶段按实验点记录的实际通气条件作为边界，不强制把边界流量改写为 `5.0 / 2.4 / 1.8` 等名义计量比；这些标准计量比只用于模型标定完成后的工况扫描。

## 模型结构

1. 台架供给边界：按测试点给定电堆入口总流量、温度、压力、湿度和新鲜空气组分；标定阶段最终入堆湿度按实验点调理，不把回流混合造成的湿度偏差再作为拟合目标。
2. EGR回流边界：按循环比从电堆阴极出口气中取回流气体，并与台架新鲜供给混合。
3. 水分离器：去除液态水，保留水蒸气和干气组分；温度压力由传感器数据或经验插值外部给定，其中出口/分离器温度在标定阶段作为台架边界，电堆内部热状态仍独立计算。
4. 阳极侧：保留为台架边界输入，不做阳极循环；EGR测试缺失阳极数据时由无EGR数据按电流插值。
5. 电堆核心：复用已有模型中完整电堆核心，外部接口适配简化台架边界。

## 电堆欧姆项与膜水口径

1. 当前简化版电堆欧姆损失采用膜电导率公式，不再以 `theta5`、`theta6`、`theta7`、`theta8` 作为主欧姆湿度关系：
   - `eta_ohm = j * (delta_PEM / sigma_PEM(lambda,T) + ASR0)`
   - `j = I / A_cell`
   - `sigma_PEM = pem_sigma_scale * (0.005193*lambda - 0.00326) * exp(1268*(1/303.15 - 1/T))`
2. `theta5`、`theta6`、`theta7`、`theta8` 属于旧经验欧姆项参数，保留仅为兼容既有参数表和旧模型口径；当前主公式使用 `pem_sigma_scale` 和 `asr0_ohm_cm2`。
3. 膜水采用准稳态口径：
   - `lambda = (lambda_eq_ca + lambda_eq_an) / 2`
   - `lambda_eq = f(a)`，`a = p_H2O / p_sat(T)`
4. 阴极和阳极水蒸气分压由各自控制体水蒸气质量守恒和理想气体状态方程计算；阳极边界第一版保持原台架湿度边界，不在本轮无加湿器阴极 CEGR 研究中额外改变。
5. 膜水迁移项按电渗拖曳和反扩散计算，总质量流率使用电池有效面积和片数换算；从阳极到阴极为正。

## 参数拟合策略

1. 无EGR完整极化曲线用于基础电堆参数拟合。
2. 阴极等效容积比例 `ca_volume_scale` 先用无EGR实测阴极进出口平均压力标定，使零维 `PV=nRT` 内部压力接近 `0.5*(p_in+p_out)`；该参数只改变等效气体库存，不改质量守恒和源项。
3. 欧姆项拟合释放 `pem_sigma_scale` 和 `asr0_ohm_cm2`；当前实验数据主要识别出膜电导率比例修正，非膜面积比电阻 `asr0_ohm_cm2` 接近零，应避免把它解释为独立可靠的硬件内阻测量。
4. EGR稳态点只用于EGR敏感参数、边界修正参数和经验插值参数拟合。
5. Nernst形式、法拉第耗氧产水关系、质量守恒、能量守恒和组分混合关系不作为自由拟合项。

## 验收指标

1. 无EGR极化曲线：单片电压误差、低中高电流段误差、入口/出口边界输入一致性，以及内部压力相对实测进出口平均压力的偏差。
2. EGR趋势：0.1 A/cm2下循环比升高时电压下降趋势可复现；0.05 A/cm2补充点作为弱验证。
3. 入口组分：入堆氧摩尔分数和按实验边界反算的氧计量比作为诊断量；标定时以实际通入气体条件为准，不反向约束到名义计量比。
4. 守恒检查：氧消耗、氮守恒、水生成/去除/回流闭合、循环比和单位换算链一致。

## 执行步骤

1. 检查已有车载/台架模型中可复用电堆核心的接口。
2. 生成最小数据整理和初始化脚本。
3. 新建简化版 Simulink 模型，建立台架供给、EGR混合、水分离、阳极边界、电堆核心和诊断输出。
4. 运行模型结构检查和初始化脚本检查。
5. 如模型可运行，进入基础标定；如接口不匹配，先修正边界适配，不改电堆内部方程。
