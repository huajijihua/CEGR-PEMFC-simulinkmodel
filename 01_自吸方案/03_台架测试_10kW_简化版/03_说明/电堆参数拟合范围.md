# 简化台架电堆参数拟合范围

本文用于区分 `PEMFCStackCore` 中哪些量是不可调物理常数，哪些量可作为后续标定参数。当前结论是：电压参数只是第一组，温度、湿度、压力链路异常时，应回到电堆结构、出口导纳、热参数、膜水参数和边界映射中找原因。

## 1. StackModelParam_simplified

| 索引 | 参数 | 当前值来源 | 主要影响 | 当前分类 |
|---:|---|---|---|---|
| 1 | `N_cell` | 电堆实物片数 | 电压、反应消耗、产热 | 固定实物参数 |
| 2 | `A_cell_cm2` | 电堆有效面积 | 电流密度、反应速率、热/水通量 | 固定实物参数，除非有效面积证据变化 |
| 3 | `V_ca_m3` | 阴极气体库存体积 | 阴极压力动态、气体库存、膜水活度 | 可调结构参数 |
| 4 | `V_an_m3` | 阳极气体库存体积 | 阳极压力动态、膜水活度 | 可调结构参数 |
| 5 | `K_ca_out_kg_s_kPa` | 阴极出口等效导纳 | 阴极出堆流量、内部压力、停留时间 | 可调气路参数 |
| 6 | `K_an_out_kg_s_kPa` | 阳极出口等效导纳 | 阳极出堆流量、阳极库存 | 可调气路参数 |
| 7 | `C_stack_J_K` | 电堆等效热容 | 温度动态、稳态温度响应 | 可调热参数 |
| 8 | `h_cool_W_K` | 默认冷却换热 | 冷却散热、堆温 | 可调热参数 |
| 9 | `h_amb_W_K` | 环境换热 | 环境散热、低负荷温度 | 可调热参数 |
| 10 | `E_nernst_ref_V` | Nernst 基准 | 开路/热力学电压 | 暂固定，不优先拟合 |
| 11 | `E_nernst_temp_coeff_V_K` | Nernst 温度项 | 温度对可逆电压影响 | 暂固定，不优先拟合 |
| 12 | `ASR0_ohm_cm2` | 标定 CSV | 非膜本体欧姆损失 | 电压拟合参数 |
| 13 | `j0_c_A_cm2` | 标定 CSV | 阴极活化极化 | 电压拟合参数 |
| 14 | `conc_loss_c` | 默认值/接口占位 | 浓差极化系数 | 当前不拟合，etaCon=0 |
| 15 | `iL_A_cm2` | 默认值/接口占位 | 极限电流密度 | 当前不拟合，etaCon=0 |
| 16 | `sigma_pem_correction` | 标定 CSV | 膜电导率修正，欧姆极化 | 电压/膜水耦合拟合参数 |
| 17 | `alpha_O2` | 标定 CSV/默认值 | 阴极电荷转移系数，活化极化 | 第二轮电压拟合参数 |
| 18 | `thermoneutralVoltage_V` | 热中性电压参考 | 反应热 `Qgen` | 暂固定，必要时热模型审查 |
| 19 | `tau_mem_s` | 标定 CSV/默认值 | 膜水通量动态松弛 | 可调膜水动态参数，但稳态电压阶段不拟合 |

## 2. PEMFCStackCore 内部固定常数

这些值直接写在 MATLAB Function 中，不通过 `StackModelParam_simplified` 输入：

```text
alpha_H2 = 0.5
I0_a = 0.1 A/cm2
I_leak = 0.01 A/cm2
delta_PEM = 0.0025 cm
rho_PEM = 1980 kg/m3
EW = 1.1 kg/mol
```

`alpha_O2` 已因第一轮拟合不合格而开放为拟合参数，默认参考值为 `0.3`，当前拟合范围为 `0.15..1.00`。其余常数当前仍不作为拟合参数。若后续有明确材料、泄漏电流、膜厚或阳极动力学证据，可以单独解锁，但不能和当前电压参数一起盲目放开。

## 3. CoolingCurveParam_simplified

冷却曲线不是 `stackModel` 的一部分，但直接决定 `hCoolEff`，因此温度误差不应只靠电压参数解释。

| 参数 | 主要影响 | 当前分类 |
|---|---|---|
| `cool_flow_curve_enabled` | 是否按冷却流量插值换热系数 | 固定开关，通常保持启用 |
| `cool_flow_curve_L_min` | 冷却流量断点 | 来自实验/标定曲线，暂固定 |
| `cool_flow_curve_h_W_K` | 各流量下等效换热系数 | 可调热参数 |

电堆热平衡求解的是 `T_stack`，不是直接求出口气体温度。当前采用方案 B：`T_stack` 按阴极入口/出口平均温度解释，温度标定默认用 `(stack_in_T_C + stack_out_T_C)/2` 作为目标；阴极出口温度由 `T_ca_out = 2*T_stack - T_ca_in` 估算并作为诊断输出。

最近一次电压回放中，压力边界误差为 0 是因为入口/出口压力直接由实验边界给定；但出口温度误差较大，no-EGR 约 `5.10 C RMSE`，EGR 约 `9.55 C RMSE`。因此已新增 `calibrate_testbench_10kw_simplified_temperature.m`，优先标定 `cool_flow_curve_h_W_K`。

温度标定结果写入 `00_输入参数/标定参数/simplified_thermal_params.csv`，验证表位于 `04_验证结果/temperature_fit_v01`。当前方案 B 结果为：

```text
T_stack target RMSE: 4.19 C -> 0.65 C
stack_out_T RMSE:   8.37 C -> 1.29 C
```

注意：5.5 L/min 工况多为低负荷，目标堆温低于或接近冷却入口温度，反推得到的等效换热能力约 `7.47 W/K`，物理含义更接近“该低流量/低负荷条件下冷却水对堆温的有效约束很弱”，不应直接外推为冷却系统真实换热能力。`temperature_fit_h_required_by_flow.csv` 还标记了 6.5、9.1、11.1 L/min 节点相对前一流量下降，说明直接按实验点拟合会出现非单调曲线；后续若要求物理单调性，需要重新启用单调约束并接受相邻节点平台化。

## 4. CaseBoundaryParam_simplified

`CaseBoundaryParam_simplified` 多数来自实验表，原则上不是模型拟合参数，而是工况边界。若温湿压不对，先判断它属于“边界输入错误”还是“模型响应错误”。

| 索引 | 量 | 来源 | 是否拟合 |
|---:|---|---|---|
| 1 | 电流 | 实验表 | 不拟合 |
| 2:7 | 阴极入口压力/温度/流量/组分 | 实验表和湿空气换算 | 不拟合，先审查数据映射 |
| 8 | 阴极出口背压 | 实验表 `stack_out_p_kPa` | 不拟合，当前为压力边界 |
| 9:11 | 分离器温压和 EGR 比例 | 实验表 | 不拟合，先审查 EGR 链路 |
| 12:15 | 阳极入口和背压 | 实验表和氢气湿度换算 | 不拟合，先审查数据映射 |
| 16:17 | 冷却入口温度和流量 | 实验表 | 不拟合，作为热模型边界 |

如果某个量是实验边界，拟合不能改变它；如果该量的模型输出仍有误差，才回到对应的结构参数、导纳、热参数或膜水参数中查找。

## 5. 电压拟合当前状态

1. 第一轮含浓差项拟合未通过物理性审查；扩大范围后 `conc_loss_c`、`iL_A_cm2` 均贴上界，且 `etaCon` 在当前电流范围内量级很小，因此当前版本暂不考虑浓差极化，令 `etaCon = 0`。
2. 当前四参数拟合范围为：`ASR0_ohm_cm2 0..0.50`、`j0_c_A_cm2 1e-10..1e-3`、`sigma_pem_correction 0.05..1.50`、`alpha_O2 0.15..1.00`。
3. 当前无浓差四参数拟合结果数值上通过第一门槛：`no-EGR RMSE = 18.33 mV`、`no-EGR max abs = 27.93 mV`，但 EGR 验证仍为 `RMSE = 36.70 mV`、`max abs = 65.22 mV`。
4. 当前结果仍只能作为候选：`ASR0`、`j0_c_A_cm2` 贴下界，`sigma_pem_correction` 贴上界，只有 `alpha_O2` 未贴边。多起点扫描表 `voltage_fit_multistart_scan.csv` 显示多个初值收敛到同一类边界解，因此这不是单一起点导致的局部最优问题。
5. 后续不应继续盲目扩大电压参数集。应先用 `voltage_fit_noegr_residuals.csv`、`voltage_fit_egr_validation.csv`、`voltage_fit_metrics.csv` 检查温湿压残差、欧姆面积电阻和 EGR 趋势。
6. 若温度误差主导，先调热参数：`C_stack_J_K`、`h_cool_W_K`、`h_amb_W_K`、`cool_flow_curve_h_W_K`。
7. 若压力/流量链路出错，先调气路结构参数：`V_ca_m3`、`V_an_m3`、`K_ca_out_kg_s_kPa`、`K_an_out_kg_s_kPa`。
8. 若 RH、膜水和欧姆项耦合异常，再考虑 `tau_mem_s` 和膜水相关模型，不先解锁材料常数。

## 6. 压力标定状态

已新增 `calibrate_testbench_10kw_simplified_pressure.m`，目标是把 `PEMFCStackCore.summary(5)` 的 `pCaDiag` 和 `summary(7)` 的 `pAnDiag` 分别拟合到实验进/出口绝压平均值：

```text
pCa_target = 0.5 * (stack_in_p_abs_kPa + stack_out_p_abs_kPa)
pAn_target = 0.5 * (anode_in_p_abs_kPa + anode_out_p_abs_kPa)
```

当前压力标定结果写入 `00_输入参数/标定参数/simplified_pressure_params.csv`，验证表位于 `04_验证结果/pressure_fit_v01`。当前候选参数为：

```text
V_ca_m3             2.0e-4 -> 8.0e-4
V_an_m3             1.5e-4 -> 1.5e-4
K_ca_out_kg_s_kPa   1.2e-4 -> 3.17005332950444e-4
K_an_out_kg_s_kPa   6.67e-6 -> 2.195302844285e-5
```

压力回放指标：

```text
pCa RMSE: 11.94 kPa -> 2.96 kPa (K-only) -> 1.73 kPa (V_ca opened)
pAn RMSE:  8.17 kPa -> 0.75 kPa (K-only) -> 0.73 kPa (final)
```

注意：EGR 低负荷点压力误差已经较小，`pCa RMSE` 约 `0.23 kPa`；主要残差来自 no-EGR 高电流段，最大 `pCa` 误差仍约 `5.91 kPa`。因此当前压力参数是候选值，不应视作最终定版；下一步应检查高电流段阴极出口流量公式、单一 `K_ca` 是否足够，以及是否需要流量相关出口导纳。
