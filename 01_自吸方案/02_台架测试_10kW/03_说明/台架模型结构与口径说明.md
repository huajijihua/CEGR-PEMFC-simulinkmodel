# 台架模型结构与口径说明

更新时间：2026-06-07

## 当前模型入口

```text
01_模型/CEGR_TestBench_10kW_v01.slx
02_脚本/init_testbench_10kw_v01.m
02_脚本/calibrate_testbench_10kw_v01_thermal_stageA.m
02_脚本/run_testbench_10kw_v01_constant_current.m
```

第一版台架模型从 10 kW GZS60 车载模型复制并收口，复用电堆核心、压力链、热侧和电压参数；删除膜加湿器。台架入口湿度由台架空入端设定，空压机位置由 DQ60 氢气循环泵等效代替。

## 主气路

```text
BenchAirSupply + EGRFeedbackDelay
-> EGRMixer
-> DQ60MapCompressor
-> BenchCathodeConditioner
-> PEMFCStackCore
-> ThreeWayValve
   -> bench_out_node（台架空出端边界，不再反馈系统）
   -> WaterSeparator
   -> EGRValve
   -> EGRFeedbackDelay
```

台架空入端位于 DQ60 前。台架给定气体组分、温度、压力和湿度；EGR 回流也在 DQ60 前混合。

## DQ60 空压机等效模型

`DQ60MapCompressor` 采用书籍《质子交换膜燃料电池系统及其控制》中更适合系统级仿真的空压机口径：

```text
入口混合气质量流量/组分/T/p
-> 换算入口体积流量
-> DQ60 map 查压升
-> 等熵压缩 + 效率修正计算出口温升和功耗
```

当前参数来源：

- DQ60 供应商 PDF：`4_DQ60氢气循环泵MAP图 - 英文.pdf`。
- 第一版仅使用 `Air medium`、进气 150 kPa 页面的流量特性曲线，按图上点近似数字化。
- 当前默认转速为 `3000 rpm`，后续应由台架实测转速或控制命令替换。
- 当前效率默认 `0.62`，用于出口温升和功耗量级估算，尚未由功率 map 标定。

由于原始稳态数据没有台架源端压力和 DQ60 转速，初始化脚本会根据实验阴极进堆压力反算一个 DQ60 入口压力，使无 EGR 基准点能保持原实验入堆压力。这个压力是模型反算量，不是实测量。

## 循环比定义

当前循环比指令按三通前电堆阴极出口气相总流量定义：

```text
alpha_EGR_cmd = m_return_branch_gas / m_stack_cathode_out_gas
alpha_EGR_actual = m_EGR_return_gas_after_separator / m_stack_cathode_out_gas
```

第一版水汽分离器不损失 O2/N2，也不主动降温；若没有冷凝，`alpha_EGR_actual` 与 `alpha_EGR_cmd` 基本一致。

## 流量控制口径

恒电流扫描采用与车载模型一致的固定总进空压机流量口径：

```text
m_compressor_total = m_cathode_flow_from_bench_case
m_bench_air_in = m_compressor_total - m_EGR_return
```

台架空入端的新鲜气体温度、压力保持对应基准工况设定值不变；其中压力为 DQ60 入口压力，当前由初始化脚本按无 EGR 实验入堆压力反算。

湿度口径按实验“电堆入口”条件处理，而不是按 DQ60 入口压力下的相对湿度处理。初始化脚本先用原始台架无 EGR 数据中的 `阴极进堆温度_C`、`阴极进堆压力_kPa` 和 `阴极进堆湿度_pct` 计算水蒸气摩尔分数，再换算 O2/N2/H2O 质量流。若 DQ60 和后续温控段不发生冷凝，气体中的水蒸气质量流保持不变；温控回到实验入堆温度后，电堆入口 RH 应回到实验值。

## 台架调控装置

`BenchCathodeConditioner` 是第一版等效温控装置，用于代表 DQ60 后到电堆阴极入口之间的台架温度调控能力：

- 出口温度取原台架稳态数据 `阴极进堆温度_C`。
- 出口压力不再被该块强制设定，沿用 DQ60 出口压力。
- 不改变 O2、N2、H2O 质量流量；若水蒸气超过当前温压饱和量，则转为液态水诊断。

## 水汽分离器

`WaterSeparator` 只作用在三通阀后的 EGR 回流支路。第一版温度口径：

```text
T_separator = T_stack_cathode_out
```

处理规则：

- O2/N2 不变。
- 已有液态水全部排出。
- 当前温压下过饱和的水蒸气冷凝并排出。
- EGR 回流出口不携带液态水。

## 热模型 Stage A 校准

台架模型继续使用车载侧 `PEMFCStackCore` 的集总热模型：

```text
Q_net = Q_gen - Q_cool - Q_amb - Q_gas
Q_cool = h_cool_eff(coolant_flow_L_min) * (T_stack - T_cool_in)
```

本轮补充了台架专用热校准入口：

```text
02_脚本/calibrate_testbench_10kw_v01_thermal_stageA.m
```

校准口径：

- 使用 13 个无 EGR 台架稳态点。
- 冷却液散热目标按 `rho * cp * flow * (T_cool_out - T_cool_in)` 计算，诊断表保留带符号热量；拟合曲线使用散热量幅值。
- 当前接受 `T_ca_out = T_stack + 3 C` 作为第一版工程关系，因此热校准中用 `T_stack_fit = T_ca_out_bench - 3 C` 反推热驱动力。
- `T_stack_fit - T_cool_in < 2.5 C` 的低热驱动力点不参与曲线拟合，只保留为诊断点，避免小温差放大 `h_cool` 噪声。
- 台架目录存在 `00_输入参数/标定参数/testbench_thermal_stageA_*` 文件时，`init_testbench_10kw_v01.m` 会优先读取台架热曲线，不覆盖车载侧热标定文件。

## 当前验证

已运行：

```text
run_testbench_10kw_v01_constant_current([0 0.1 0.2 0.3 0.4])
```

输出：

```text
04_验证结果/testbench_constant_current_egr_scan.csv
```

共 65 行，即 13 个基准电流点 x 5 个循环比。结果文件包含 DQ60 入口压力、压升、入口体积流量、出口温度、功耗、map 钳位诊断，以及 `T_stack_C`、`Q_gen_W`、`Q_cool_W`、`Q_amb_W`、`Q_gas_W`、`h_cool_effective_W_K` 等热诊断列。

当前回跑结果：

- 无 EGR 13 点单片电压 MAE 约 5.48 mV，RMSE 约 7.07 mV。
- 无 EGR 有效冷却换热系数范围约 1155.4-1636.9 W/K，说明台架热曲线已被模型加载。
- 无 EGR `Q_cool_W` 相对冷却液带符号散热量的 RMSE 约 928.9 W；低负荷点冷却液温差符号和小热驱动力仍应作为台架数据质量问题复核。
- 按修正后的湿度口径，38 A 无 EGR 点电堆入口 RH 为 0.800，温控段没有冷凝液态水。

## 后续待实验确认

- 台架空入端压力/DQ60 入口压力是否有实测值。
- DQ60 转速是否有实测值或控制命令。
- DQ60 map 是否有原始数据表，当前图像数字化点需要替换为精确数据。
- DQ60 后温控装置是否确有对应硬件，是否应计入压降或功耗。
- 水汽分离器是否有出口温度、压降、排水量或分离效率数据。
- 低负荷冷却液进出口温差符号、温度传感器位置和冷却液流量可信度。
- cEGR 台架实测循环比、流量计位置和测量口径。
- 恒电流、恒电压、恒入口氧分压工况扫描和 MATLAB 可视化已完成初版；其中恒电压电流密度反求保留三位小数。
- 恒入口氧分压当前分为两类结果：legacy `condition_study_constant_pO2_inlet_solved.csv` 为等效流量倍率扫描；DQ60 口径结果使用 `run_testbench_10kw_v01_pO2_DQ60_study.m` 在 `air_flow_scale` 和 DQ60 转速内搜索稳定可行补偿点。
- 0.1 A/cm2、EGR=0.25 的统一口径两点结果为 `air_flow_scale=1.5`、DQ60 约 86 L/min、`pO2_ca_in` 比无 EGR 基准低约 1.10 kPa；该点为稳定可行但未完全达到恒氧分压目标。早期 `flowScale=4 / 4000 rpm` 点接近 pO2 目标但存在动态振荡，不作为稳态代表结论。
